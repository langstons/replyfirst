# Installation Guide for ReplyFirst

## Quick Start

### 1. Load the Extension in Chrome

1. Open Chrome browser
2. Navigate to `chrome://extensions/`
3. Enable **Developer mode** (toggle in the top-right corner)
4. Click **Load unpacked**
5. Navigate to and select the `/Users/brent.langston/git/replyfirst` directory
6. The ReplyFirst icon should appear in your Chrome toolbar

### 2. Configure Gmail

1. Open Gmail (https://mail.google.com)
2. Click the gear icon ⚙️ in the top-right
3. Select **See all settings**
4. Under the **General** tab, find **Conversation View**
5. Ensure it's set to **Conversation view on**
6. Click **Save Changes** at the bottom

### 3. Enable ReplyFirst

1. Click the ReplyFirst icon in your Chrome toolbar
2. Toggle the switch to **ON** (it should turn purple)
3. Refresh your Gmail tab

### 4. Verify It's Working

1. Open any email conversation with multiple messages
2. The newest message should now appear at the top
3. The oldest message should be at the bottom

## Troubleshooting

### Extension Not Working?

1. **Check Gmail Settings**: Ensure Conversation View is enabled
2. **Refresh Gmail**: Press Ctrl+R (Cmd+R on Mac) to reload
3. **Check Extension Status**: Click the ReplyFirst icon and verify it's enabled
4. **Clear Cache**: Try clearing your browser cache
5. **Reload Extension**: Go to `chrome://extensions/`, find ReplyFirst, and click the reload icon

### Toggle Not Responding?

1. Close and reopen the popup
2. Refresh the Gmail tab after toggling
3. Check the Chrome console for any errors (F12)

### Messages Still in Wrong Order?

1. Make sure you're viewing a conversation (not individual emails)
2. Try disabling and re-enabling the extension
3. Hard refresh Gmail with Ctrl+Shift+R (Cmd+Shift+R on Mac)

## Uninstalling

1. Go to `chrome://extensions/`
2. Find ReplyFirst
3. Click **Remove**
4. Confirm the removal

## Privacy & Permissions

ReplyFirst only requests the following permission:
- **storage**: To save your enable/disable preference

The extension:
- Does NOT collect any personal data
- Does NOT track your browsing
- Does NOT make external network requests
- Only runs on Gmail pages (mail.google.com)
