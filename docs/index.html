Get Started in Minutes

Install ReplyFirst and start seeing your newest messages first

1

Download the Extension

Get the latest version from our GitHub releases page. The extension is packaged and ready to install.

2

Install in Chrome

Open Chrome, go to chrome://extensions/, enable Developer Mode, and load the unpacked extension.

3

Enable the Toggle

Click the ReplyFirst icon in your toolbar and turn on the toggle switch. That's it!

4

Open Gmail

Visit Gmail and enjoy seeing your newest messages first. Make sure conversation view is enabled in Gmail settings.
