ReplyFirst collects ZERO data. We believe privacy is a fundamental right, not a feature. This extension operates entirely locally in your browser with no external connections.

What Data We Collect

None. ReplyFirst does not collect, store, track, or transmit any user data whatsoever.

How the Extension Works

ReplyFirst uses a simple CSS injection technique to reverse the visual order of Gmail conversation threads. Here's exactly what happens:

  • Local Storage: Your enable/disable preference is stored locally in your browser using Chrome's storage API. This data never leaves your device.
  • CSS Injection: When enabled, the extension injects CSS styling into Gmail pages to reverse the display order using flexbox.
  • No Reading: The extension does not read, access, or process any email content.
  • No Transmission: Zero network requests are made by this extension.

Permissions Explanation

Storage Permission

Used exclusively to save your enable/disable preference locally on your device. No data is synchronized or transmitted.

Host Permission (https://mail.google.com/*)

Required to inject CSS styling into Gmail pages. The extension only runs on Gmail and does not access any other websites.

Third-Party Services

ReplyFirst does not use any third-party services, analytics, tracking, or external libraries. The extension is completely self-contained.

Data Security

Since we don't collect any data, there is no data to secure, breach, or compromise. Your email content remains entirely within Gmail's infrastructure and is never touched by ReplyFirst.
